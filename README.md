# Association_Rule
Association Rule Mining project using the Apriori algorithm to discover frequent itemsets and meaningful relationships from transactional datasets. Includes data preprocessing, transaction encoding, rule generation, and evaluation using Support, Confidence, Lift, Leverage, and Conviction to derive actionable business insights and recommendations.
